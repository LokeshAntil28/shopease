# ShopEase

A simple front-end shop UI (single `index.html`) with:
- product list + search
- category filter + sorting
- cart sidebar with quantity controls
- checkout message
- cart saved in your browser (`localStorage`)

## Open it
Open `index.html` in your browser.

## Notes
- No backend is used.
- Cart data stays in your browser (via `localStorage`).
